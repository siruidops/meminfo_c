tested on Arch linux + GCC 10.2.0 + GNU Make 4.3

Dependencies:
```
make
gcc
```

Build and install:
``` bash
$ make
$ sudo make install
```

Clean:
```
$ make clean
```

Uninstall:
```
$ sudo make uninstall
```
